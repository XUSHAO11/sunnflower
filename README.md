# Vue 3 + Vite + ant-design + pinia

安装方式：

```js
git clone https://github.com/XUSHAO11/Sunflower.git
```

```js
npm install && yarn
```

```js
npm run dev && yarn dev
```

## 使用工具

- 采用markdown-it转换成可识别样式
- animate.css 动画效果
- ant-design v3组件库
- 留言采用 Valine
